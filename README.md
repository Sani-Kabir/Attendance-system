Student Attendance System (Flask + HTML + CSS + JS)

This is a simple and clean Student Attendance System built using:

Python (Flask)

HTML

CSS

JavaScript

It allows users to:

Add a student's name

Automatically record attendance with date & time

Display all attendance records

Store all data in a JSON file

Run in any computer with Python installed

Features

 Mark attendance instantly
 Beautiful and responsive UI
 Displays attendance table dynamically
 Works with a simple API


 Technologies Used
Component	Technology
Backend API	Flask (Python)
Frontend	HTML + CSS
Logic/Interaction	JavaScript (Fetch API)

Database	JSON file (attendance.json)
